## Что это

Главная точка входа для поиска любых расширений [[Claude Code]] ([GitHub](https://github.com/hesreallyhim/awesome-claude-code)). Содержит категоризированные ссылки на сотни инструментов.

---

## Основные категории

- **Agent Skills** -- специализированные конфигурации (DevOps, security, fullstack)
- **Workflows & Knowledge Guides** -- наборы ресурсов под проекты
- **Tooling** -- инструменты поверх Claude Code:
  - **Auto-Claude** -- автоматизация задач
  - **Claude Squad** -- управление несколькими агентами
  - **Claude Swarm** -- multi-agent orchestration
  - **Ralph** -- autonomous task loops
- **Config Managers** -- claude-rules-doctor (валидация), ClaudeCTX (переключение конфигов)
- **Usage Monitors** -- ccflare (web-dashboard), CC Usage (CLI аналитика)
- **IDE Integrations** -- VS Code, Emacs, Neovim
- **Status Lines** -- Rust-реализации с git/token tracking
- **Alternative Clients** -- альтернативные интерфейсы
