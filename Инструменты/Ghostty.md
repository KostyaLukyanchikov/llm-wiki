# Ghostty

> GPU-ускоренный терминал от Митчелла Хашимото (HashiCorp) с нативным UI и минимальным потреблением памяти. ([GitHub](https://github.com/ghostty-org/ghostty) | [ghostty.org](https://ghostty.org))

Назад: [[README]] | Папка: `Инструменты/`

---

## Что это

Терминальный эмулятор с GPU-ускорением через Metal (macOS), созданный Митчеллом Хашимото (создатель Vagrant, Terraform, HashiCorp). Сочетает нативный platform UI с высокой производительностью рендеринга. Написан на Zig.

---

## Почему для LLM-разработки

| Параметр | Ghostty | VS Code (2 проекта) |
|---|---|---|
| Потребление RAM | ~500 MB | ~8 GB |
| Стабильность памяти | Не растёт в длинных сессиях | Растёт с расширениями |
| Нативные сплиты | Да, встроенные | Через terminal tabs |

Ghostty не течёт по памяти в длинных agentic-сессиях -- критично при многочасовой работе с Claude Code.

### Паттерн: Ghostty + Claude Code + Neovim

```
┌──────────────────┬──────────────────┐
│                  │                  │
│   Claude Code    │     Neovim       │
│                  │                  │
│                  │                  │
├──────────────────┴──────────────────┤
│           claude-monitor            │
└─────────────────────────────────────┘
```

Ghostty в сплитах полностью заменяет IDE: Claude Code для агентной работы, Neovim для ручного редактирования, нижний сплит для мониторинга.

---

## Ключевые возможности

- GPU-ускоренный рендеринг через Metal (macOS) / OpenGL (Linux)
- Нативный UI платформы (не Electron)
- Встроенные сплиты и табы
- Конфигурация через текстовый файл `~/.config/ghostty/config`
- Быстрый шрифтовой рендеринг с поддержкой лигатур
- Минимальное и стабильное потребление памяти

---

## Установка

```bash
brew install --cask ghostty
```

### Базовая конфигурация

Файл `~/.config/ghostty/config`:

```
font-family = JetBrains Mono
font-size = 14
theme = catppuccin-mocha
window-padding-x = 8
window-padding-y = 8
```

## Горячие клавиши (сплиты)

| Клавиша | Действие |
|---|---|
| Cmd+D | Вертикальный сплит |
| Cmd+Shift+D | Горизонтальный сплит |
| Cmd+[ / Cmd+] | Переключение между сплитами |
| Cmd+W | Закрыть сплит |
