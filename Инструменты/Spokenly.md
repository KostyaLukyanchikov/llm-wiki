# Spokenly

> AI-диктовка для macOS/iOS -- голос в чистый текст с пунктуацией в любом приложении. ([docs](https://spokenly.app/docs) | [App Store](https://apps.apple.com/app/id6740315592))

Назад: [[README]] | Папка: `Инструменты/`

---

## Что это

Приложение для macOS и iOS, которое превращает голос в текст с автоматической пунктуацией. Работает в любом приложении через системный ввод. Включает Agent Mode для голосового управления Mac и deeplinks для интеграции со скриптами.

---

## Ключевые возможности

- Диктовка в любое текстовое поле macOS с автопунктуацией
- Agent Mode -- голосовое управление Mac:
  - Запуск приложений
  - Выполнение shell-команд
  - Ask Claude (запрос к Claude через голос)
  - Google Search голосом
- Deeplinks для автоматизации: `spokenly://start`, `spokenly://toggle`
- Local Only Mode -- полностью офлайн на локальных моделях
- Pro-подписка через App Store -- безлимитные online-модели

---

## Интеграция с Claude Code

Диктовка промптов голосом прямо в терминал с Claude Code. Вместо набора длинных текстовых промптов -- наговори задачу, Spokenly превратит в текст с пунктуацией.

### Deeplinks для hooks и скриптов

```bash
# Запуск диктовки
open "spokenly://start"

# Toggle диктовки
open "spokenly://toggle"
```

Deeplinks можно использовать в Claude Code hooks или shell-скриптах для триггера диктовки по событию.

## Agent Mode

Agent Mode превращает Spokenly из диктовки в голосовой интерфейс управления Mac:

- **Запуск приложений**: "Open Terminal", "Open Safari"
- **Shell-команды**: выполнение команд через голос
- **Ask Claude**: отправка запроса к Claude и озвучивание ответа
- **Google Search**: голосовой поиск
